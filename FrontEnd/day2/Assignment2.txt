Day 2 Assignments
1. Create a Registration Form
    Include fields for name, email, password, gender, hobbies, and a submit button.
    Make the email and password fields required.

2. Create a Blog Page
    Use semantic elements (<header>, <article>, <footer>).
    Add a navigation bar and an article with a title and description.

3. Create a Survey Form
    Use checkboxes, radio buttons, and a dropdown menu.
    Add validation for required fields.

4. Design a Webpage Layout
    Use <header>, <main>, <aside>, and <footer> to create a simple layout.
    Include links, lists, and a navigation bar.
